• Introduction: Small tools description
  • Tests: Run some function test of in2publish
  • Show configuration: Display the local configuration of in2publish from LocalConfiguration.yaml
  • Show logs: Show logfiles of in2publish
  • Inspect TCA: See which parts of the TCA are processed, and the reason why others are not.
  • Flush TCA cache: Immediately flush all cached TCA entries.
  • Remove superfluous envelopes: Deletes all envelopes which were already used (envelopes are one-way records).
  • remove all registry entries: removes all registry entries which store information about the local and foreign environment
